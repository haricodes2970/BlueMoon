# BlueMoon

BlueMoon is a long-term communication platform. The first product built on the
platform is **PINChat**.

This repository is currently in the **initialization phase**: repository
structure, documentation, and engineering standards are being established
before application code is written.

## Status

🚧 Pre-implementation — see [ROADMAP.md](./ROADMAP.md) for current phase and
progress.
